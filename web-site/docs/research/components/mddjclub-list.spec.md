# MddjclubList Specification

## Overview

- **Target file:** `src/app/list/page.tsx`
- **Screenshot:** `docs/design-references/mddjclub.com/list-desktop.png`
- **Interaction model:** static filters + grid in captured state

## Sections

### Header + breadcrumb

- Reuses the homepage shell
- Breadcrumb sits below the fixed header

### Filter panel

- Large rounded white card
- Each filter row uses a left label and chip list on the right
- First chip in each row is blue/active

### Product listing

- Four-column desktop grid
- Product card treatment matches home cards

### Pagination

- White control bar below the grid
- Page `1` active in blue
