# MddjclubLogin Specification

## Overview

- **Target file:** `src/app/login/page.tsx`
- **Screenshot:** `docs/design-references/mddjclub.com/login-desktop.png`
- **Interaction model:** static auth screen in captured state

## Sections

### Header

- Minimal white top bar
- Left side shows logo + "欢迎登录"
- Right side shows three quiet links

### Auth shell

- Large centered rounded white container
- Desktop two-column split with blue marketing panel on the left
- Mobile removes the blue panel and keeps only the form

### Form

- Big page title
- Small registration link
- Two auth tabs with password tab active
- Two outlined inputs and one large blue submit button
