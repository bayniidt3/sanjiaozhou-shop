# MddjclubHome Specification

## Overview

- **Target file:** `src/app/page.tsx`
- **Screenshot:** `docs/design-references/mddjclub.com/home-desktop.png`
- **Interaction model:** static with fixed header

## Sections

### Header

- Utility strip on light gray background
- White main nav with logo, route links, rounded search box, and three CTA buttons
- Active route uses blue underline

### Hero

- Desktop split layout: left category card, right blue banner
- Mobile collapses to a single banner-first flow

### Market table

- White card with soft border and rounded corners
- Dense 4-column table with bold ratio values

### Featured cards

- Three-column desktop grid
- One-column mobile stack
- White rounded cards with screenshot image, discount pill, many small tags, and price CTA row

### CTA + footer

- Full-width blue gradient callout
- Dark footer with four content columns
